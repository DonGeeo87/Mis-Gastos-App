Acerca del Proyecto

MisGastos es una aplicación desarrollada con las mejores prácticas de Android, utilizando tecnologías modernas como Kotlin y Jetpack Compose. El proyecto sigue una arquitectura MVVM robusta y está diseñado para ser escalable y mantenible.

Kotlin
